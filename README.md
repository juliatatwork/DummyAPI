## Описание проекта
Данный проект был создан для тестирования API на сервисе https://dummyapi.io/. Для выполнения запросов необходим app_id, который можно получить автоматически после регистрации на самом сайте. Объектом тестирования является POST.


### POST
___
#### GET /post (GET LIST)
Возвращает список сообщений, отсортированных по дате создания
- доступен query params для вывода пагинации
- доступен query params для вывода определенного количества сообщений

Response Body:

''List''
```javascript
{
  data: Array(Model)
  total: number(total items in DB)
  page: number(current page)
  limit: number(number of items on page)
}

''Post Preview''
```javascript
{
id: string(autogenerated)
text: string(length: 6-50, preview only)
image: string(url)
likes: number(init value: 0)
tags: array(string)
publishDate: string(autogenerated)
owner: object(User Preview)
}
```
